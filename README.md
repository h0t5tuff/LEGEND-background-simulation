# LEGEND-background-simulation

designed to run on NERSC

Here I simulate the backgrounds in all LEGEND materials as detected by HPGe.
via Legend-metadata, l200-pygeom and l1000-pygeom; one can generate gdml flies with the LEGEND 200/1000 geometry and it must be placed in the approporiate sim directory.
the macro generates an lh5 file and the python script creates the energy spectrum
